# Task 1 - Data Cleaning and Preprocessing

## Objective

Clean and preprocess the Titanic dataset by handling missing values, removing duplicate records, standardizing column names, correcting data types, and removing outliers.

## Dataset

Titanic Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Steps Performed

- Loaded the Titanic dataset
- Explored dataset information
- Standardized column names
- Checked missing values
- Removed duplicate rows
- Filled missing numerical values using median
- Filled missing categorical values using mode
- Converted data types
- Removed outliers using the IQR method
- Generated descriptive statistics
