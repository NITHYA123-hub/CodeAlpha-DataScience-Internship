# Exploratory Data Analysis (EDA) - Titanic Dataset

## Objective

Perform Exploratory Data Analysis (EDA) on the cleaned Titanic dataset to understand the data, identify patterns, and visualize relationships between different features.

## Features

- Dataset Overview
- Statistical Summary
- Missing Value Analysis
- Age Distribution
- Gender Distribution
- Passenger Class Distribution
- Survival Analysis
- Correlation Heatmap

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
